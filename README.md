## About Me

I'm a **senior full‑stack web and mobile engineer** who ships end to end — from the first wireframe to the App Store release and the backend that keeps it running. My specialty is **cryptography and privacy‑first applications**: I spent four years at [**FlowCrypt**](https://flowcrypt.com) (2022–2026) as a senior engineer on their **iOS app** and **Chrome / browser extension**, shipping OpenPGP key management, passphrase‑protected keychains, Gmail/IMAP integration, and the E2EE flows users touch every day, to a production install base of privacy‑sensitive teams and regulated industries. Today I run [**Flowdesk**](https://github.com/Flowdesktech), a small remote studio building products where trust is load‑bearing.

- 🌐 **Full‑stack web** — React, Next.js, TypeScript, Node.js, Python, Firebase, Postgres. Frontend, backend, infrastructure, CI/CD — I ship whatever the product needs.
- 📱 **Mobile specialist** — Native iOS (Swift, SwiftUI) and Android (Kotlin, Jetpack Compose), plus hybrid apps with React Native, Expo, and Flutter. Real App Store and Google Play release experience, including four years at FlowCrypt.
- 🔐 **Cryptography & privacy** — OpenPGP (at FlowCrypt), AES‑GCM, Argon2id, drand/tlock identity‑based encryption, Web Crypto API, and zero‑knowledge server boundaries enforced by auditable rules. See [**Flowvault**](https://github.com/Flowdesktech/flowvault) for a public reference.
- 🧩 **Browser extension developer** — Chrome / Edge / Firefox extensions, including four years on a security‑critical E2EE extension used against real threat models.
- 🤖 **AI enthusiast & builder** — LLM orchestration, autonomous agents, RAG, evals, and automation pipelines with OpenAI, Anthropic Claude, and open‑source models. I ship AI features into real products, not demos.
- ⚡ **AI‑augmented workflow** — I use AI aggressively in the parts of engineering where it actually makes people faster: boilerplate and scaffolding, exhaustive test generation, large‑scale typed refactors, codebase exploration, and first‑pass code review. Client budgets get spent on the 20% of the code that actually needs human judgment — API design, cryptography choices, UX trade‑offs, and the gnarly edge cases — not on typing out the other 80%.
- 🌍 **100% remote** — async‑first, across time zones.
- 🧑‍💻 **Open source** — I maintain [**Flowvault**](https://github.com/Flowdesktech/flowvault), [**Firestudio**](https://github.com/Flowdesktech/firestudio), [**Snapboard**](https://github.com/Flowdesktech/Snapboard), and more under [@Flowdesktech](https://github.com/Flowdesktech).

> Currently shipping **Flowvault** — a zero‑knowledge notepad with hidden‑volume plausible deniability, drand‑backed time‑locked notes, and a Bring‑Your‑Own‑Storage local‑file mode. More privacy and crypto tools landing under [@Flowdesktech](https://github.com/Flowdesktech) soon.

## How I Work

1. **Discover** — Understand the problem, the user, the threat model, and what success actually looks like.
2. **Prototype** — Ship something clickable fast. Validate direction before scaling.
3. **Build** — Typed, tested, opinionated code, with **AI used aggressively to accelerate the boring parts** — scaffolding, boilerplate, test generation, and mechanical refactors — so that human time is concentrated on architecture, security, and the edge cases that actually matter. Cryptography choices are justified in writing, not copy‑pasted.
4. **Ship** — CI/CD, observability, and predictable releases from day one.
5. **Iterate** — Tight feedback loops with stakeholders and real users.
