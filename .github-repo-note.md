# Special "username repository"

This local folder is the source for the repository **`flowdeskadmin/flowdeskadmin`** on GitHub.

When a user on GitHub creates a repository with the **exact same name as their username**, GitHub treats it as a special "profile repository" and renders its root `README.md` on the user's profile page at https://github.com/flowdeskadmin.

## File layout

- `README.md` — this is what GitHub renders on the profile page.
- `.github-repo-note.md` — this file, just internal notes. GitHub ignores it.
- `.gitignore` — standard excludes.

## Publishing steps

1. Create a new PUBLIC repo on GitHub named **exactly** `flowdeskadmin`:
   https://github.com/new

2. Push this folder:
   ```powershell
   cd D:\Projects\flowdeskadmin-profile
   git add .
   git commit -m "Add personal profile README for @flowdeskadmin"
   git remote add origin https://github.com/flowdeskadmin/flowdeskadmin.git
   git push -u origin main
   ```

3. Visit https://github.com/flowdeskadmin — the README is rendered at the top.
